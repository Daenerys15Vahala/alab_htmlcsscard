# Blog Preview Card

## Overview

This project is a solution to the **Blog Preview Card Challenge**. The objective was to recreate the provided design as accurately as possible while building a responsive and interactive blog preview card using HTML and CSS.

The project emphasizes clean layouts, responsive design, and user interaction through hover and focus states.

- Responsive blog preview card
